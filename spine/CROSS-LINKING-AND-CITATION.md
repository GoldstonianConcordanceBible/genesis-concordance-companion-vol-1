# Cross-Linking & Citation Standard

## 1) Reference format
- Use: `Book Chapter:Verse–Verse` (e.g., `Gen 12:1–3`)
- For ranges across chapters: `Gen 11–12` (then specify focal verses)

## 2) Cross-link row rules (CSV)
Each cross-link row must include:
- source_book (Genesis)
- source_passage
- theme_tag
- target_book
- target_passage
- link_type (echo | pattern | fulfillment | contrast | doctrine | narrative)
- rationale (1–2 sentences)

## 3) Avoid long quotes
Do not paste large blocks of copyrighted translations.
Use references and summarize.

## 4) Stable IDs
- Use `book_id` like: `gcb-companion-genesis-vol-1`
- Use link IDs like: `gen-xlink-0001`

## 5) Evidence index
Each evidence entry must include:
- artifact type (sermon outline, diagram, dataset, prompt pack)
- location/path
- description
- related passages
- version
