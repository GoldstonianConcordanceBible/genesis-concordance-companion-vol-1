# Goldstonian Spine Spec (Book Companion Standard)

Every book companion must include the following sections, in order:

1) **Purpose Thesis**
- One paragraph: what the book is doing in the canon.
- One sentence: “If you miss this, you miss the book.”

2) **Narrative Flow**
- A simple arc broken into movements (3–8 movements max).
- For each movement: *what happens* + *why it matters*.

3) **Core Theology**
- 7–12 core doctrines/themes the book establishes.
- Each theme must include: definition + key references + “so what.”

4) **Messiah Thread**
- How the book anticipates, patterns, or clarifies Messiah.
- Use types, covenant promises, seed/kingdom lines, priesthood patterns, etc.

5) **Cross-Links**
- Map the book to:
  - Torah / History / Wisdom / Prophets / Gospels / Epistles
- Include: theme → references → link rationale.

6) **Hard Questions**
- 8–15 frequently asked objections or confusions.
- Each answer includes: short reply + references + “pastoral note”.

7) **Preacher’s Toolkit**
- Sermon series outlines (3–12 weeks)
- Key “turning point” passages
- Illustrations/themes that travel well
- Warnings (common misreads)

8) **Walking Rule**
- Concrete discipleship practices:
  - Home (habits)
  - Church (community)
  - Public (witness)
- Must be actionable, simple, repeatable.

## Optional add-ons
- Timelines, maps, genealogies, concept graphs
- Controlled vocabulary expansions
- Agent prompts (if you are building LLM tools)
