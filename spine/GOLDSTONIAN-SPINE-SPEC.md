# Goldstonian Spine Spec (Book Companion Standard)

Every book companion MUST include these sections, in order:

1) **Purpose Thesis**
- 1 paragraph: what the book is doing in the canon.
- 1 sentence: “If you miss this, you miss the book.”

2) **Narrative Flow**
- 3–8 movements maximum.
- Each movement: what happens + why it matters.

3) **Core Theology**
- 7–12 core themes.
- Each theme: definition + key references + “so what.”

4) **Messiah Thread**
- Patterns/types/lines that anticipate or clarify Messiah/Kingdom.
- Clearly label **pattern** vs **fulfillment**.

5) **Cross-Links**
- Dataset-driven mapping to other books:
  - Torah / History / Wisdom / Prophets / Gospels / Epistles
- Must include rationale.

6) **Hard Questions**
- 12–20 questions.
- Each: short answer + references + pastoral note.

7) **Preacher’s Toolkit**
- 6–10 week series outline
- 3–4 week “Messiah Thread” mini-series (where applicable)
- Turning-point passages
- Common misreads warnings
- Small group guide (4 sessions)

8) **Walking Rule**
- Home: 5 practices
- Church: 5 practices
- Public: 5 practices
Each practice includes: action + why + references

## Non-negotiables
- Use scripture references, not long copyrighted quotations.
- Keep language clear and practical.
- Maintain machine-friendly artifacts (CSV + schemas + evidence index).
