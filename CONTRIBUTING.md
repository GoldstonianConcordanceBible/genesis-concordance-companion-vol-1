# Contributing

## Goals
- Keep Genesis companion content aligned with the **Goldstonian Spine**
- Keep datasets **machine-friendly** and **validatable** (CSV + schemas)
- Avoid pasting large blocks of copyrighted Bible translations (use references)

## How to contribute
1. Update manuscript content in `companion/`
2. Add/adjust vocabulary entries in `vocab/controlled-vocabulary.csv`
3. Add cross-links in `data/crosswalks/genesis-crosswalk.csv`
4. Add artifacts in `data/evidence/evidence-index.csv`
5. Validate structures against `schemas/`
